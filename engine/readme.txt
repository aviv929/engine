הוראות הרצה
1.run Engine.exe
2.make sure that curpus folder and stop_words file are in the same dire

הוראות הפעלה
1.הרצת הפרוייקט ופתיחת חלון הGUI
2.בחירת תקייה המכילה את הcorpus וstop words (מחוץ לתקיית הcorpus)
3.בחירת נתיב לשמירת קובץ הpost הסופי
4.בחירת checkbox האם עם stem או לא
5. run - מפעיל ורושם זמן התחלה , בסיום מקפיץ חלון info (גדלי קבצים וכמות מילים) ומוסיף לכפתור זמני סיום
6.show info ניתן לפוח מחדש את הinfo אם סגרנו את החלון שקפץ
7.show dictionary מראה את המילון שנוצר - ניתן להפעיל רק אחרי ריצה או אחרי טעינת מילון 
8.show cache - מראה את מבנה הcache.
9.clear מנקה את כל הקבצים.
10. save cahce and dictionary - בחירת נתיב לשמירת המילון והcache
11. load cahce and dictionary - בחירת תקייה שמכילה את הקבצים. 
רק לשים לב שאם רוצים מילון עם stem  אז לפני צריך לסמן בcheckbox
